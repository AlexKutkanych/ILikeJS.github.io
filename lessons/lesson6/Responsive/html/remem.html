Размер заголовка должен быть 24px, в ссылки - 20px;

target ÷ context = result

Расчет размера текста REM ссылка

html{font-size: 100%;} т.е. 16px

h1 зададим 1.5rem = 16 * 1.5 = 24px

Ссылка находится внутри блока заголовка.
target = 20
context = 16
result = 20/16=1.25
Ссылке задаем 1.25rem, т.к. % берется от font-size <HTML>>

Расчет размера текста EM ссылка

html{font-size: 100%;} т.е. 16px

h1 зададим 1.5em = 16 * 1.5 = 24px

Ссылка находится внутри блока заголовка.
target = 20
context = 24
result = 20/24=0.8333333333333333
Ссылке задаем 0.8333333333333333em, т.к. % берется от font-size <H1>>
