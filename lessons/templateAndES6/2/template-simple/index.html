Бабушкин рецепт:
шаблон из одной строки.

  • 1) Взять яйцо
    переменая:
    var user = "Egg"
  • 2) Положить яйцо в кипящую воду
    вставить переменную в строку :
    var str = "<h1>" + user + "</h1>";
  • 3) Найти блюдо
    Найти элемент в HTML куда надо добавить этот текст: var elem = document.getElementById("cool");
  • 4) Добавить соль по вкусу
    добавить строку в HTML:
    elem.innerHTML = str;
